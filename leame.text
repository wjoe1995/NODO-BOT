instalar la libreria virtualenv
pip install virtualenv

ejecutar el siguiente comando 
virtual env

iniciar el entorno virtual desde windows
.\env\Scripts\activate 

para salir del entorno virtual escribir en consolo 
"deactivate"

para instalar requeriments.txt en caso de ser necesario
pip install -r requirements.txt

crear el archivo de dependencias
pip freeze > requirements.txt
